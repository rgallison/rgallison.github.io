Ashland Cherryland's FamFest Website

A wordpress website with a custom theme to promote Ashland's annual FamFest Run/Walk/Roll event.
Company: Alameda County

About This Project

Developed a company website using Wordpress. Implemented a custom theme with custom shortcodes to maintain the CMS capabilities. Features include dynamic content reveal, thumbnail gallery with filters and popups, cool, geometric layouts, responsiveness, interactive “Agenator” for company profile, cool hover effects, parallax headers, dynamic content interaction.
