Digital Experience Platform

A plaftorm designed to help clients create customizable, turnkey digital experiences to promote their busines, engage their customers and increase opt-ins and view analytics for each campaign.
Company: FunMobility

About This Project

Helped develop a platform designed to allow customers to customize digital experiences that could be published throught multiple channels, such as advertising, marketing, social, print, QR, etc. The platform also tracked user activity on each experience and display useful analytics to the client, including views, conversion funnels, and opt-in information. The application was developed using Play! Framework for Java and includes features such as user authentication, user permissions, content management, reporting charts.
